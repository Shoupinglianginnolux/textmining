Upload New Data

Naming Conventions for "Update New Data"
  • The file name should start with "INNOLUX Customer Support Report", with .xlsx format. The file must contain sheets named 'Cases' and 'SRQs'
  • Inside Cases sheet, there must be 'CaseNumber', 'Model', 'CreatedDate', 'SerialNumber', 'Symptom', 'Description', 'Diagnosis' columns.
  • Inside SRQs sheet, there must be 'SRNumber', 'Model', 'CreatedDate', 'SerialNumber', 'SRType', 'ErrorCode', 'InternalNotes' columns

Upload Revised Data

Naming Conventions for "Update Revise Data"
  • The file name should start with "Latest Error Data", with .xlsx format. The file must contain a sheet named 'SRQs''
  • Inside SRQs sheet, there must be 'SRNumber' and 'RevisedErrorCode' columns

Upload Keywords Mapping Table

Naming Conventions for "Update Keyword"
  • The file name should be named as "Keywords Mapping Table.xlsx", with a sheet named as 'Sheet1'
  • Inside Sheet1, there must be 'Class', 'Rule', and 'Action' columns
